API Dokumentation

Willkommen zur API-Dokumentation für das ZahlungStatistik-System. Hier erklären wir, wie du mit den verschiedenen Endpunkten interagieren kannst.

1. Erstellen einer ZahlungStatistik

Beschreibung: Erstellt eine neue ZahlungStatistik mit den angegebenen Parametern.

Beispiel Anfrage: 
POST /api?fundId=1&startDate=2023-01-01&endDate=2023-12-31
Beispiel Antwort: 
{
    "id": 123,
    "fundId": 1,
    "startDate": "2023-01-01",
    "endDate": "2023-12-31",
    "totalAmount": 10000.0,
    "averageAmount": 500.0,
    "paymentCount": 20,
    "currency": "EUR"
}

2. Alle FundStatistiken abrufen

Beschreibung: Gibt alle Funds zurück, auf die der Benutzer Zugriff hat.

Beispiel Anfrage: 
GET /api/all/fund
Beispiel Antwort: 
[
    {
        "id": 1,
        "name": "Fund A"
    },
    {
        "id": 2,
        "name": "Fund B"
    }
]

3. Alle ZahlungStatistiken abrufen

Beschreibung: Gibt eine Liste aller ZahlungStatistiken zurück.

Beispiel Anfrage: 
GET /api/all/statistics
Beispiel Antwort: 
[
    {
        "id": 123,
        "fundId": 1,
        "startDate": "2023-01-01",
        "endDate": "2023-12-31",
        "totalAmount": 10000.0,
        "averageAmount": 500.0,
        "paymentCount": 20,
        "currency": "EUR"
    }
]

4. ZahlungStatistik nach ID abrufen

Beschreibung: Ruft eine ZahlungStatistik anhand der angegebenen ID ab.

Beispiel Anfrage: 
GET /api/statistics/123
Beispiel Antwort: 
{
    "id": 123,
    "fundId": 1,
    "startDate": "2023-01-01",
    "endDate": "2023-12-31",
    "totalAmount": 10000.0,
    "averageAmount": 500.0,
    "paymentCount": 20,
    "currency": "EUR"
}

5. ZahlungStatistik erstellen oder aktualisieren

Beschreibung: Aktualisiert eine bestehende ZahlungStatistik oder erstellt eine neue, wenn keine Statistik mit der angegebenen ID existiert.

Beispiel Anfrage: 
PUT /api/statistics/123?fundId=1&startDate=2023-01-01&endDate=2023-12-31
Beispiel Antwort (Update): 
{
    "id": 123,
    "fundId": 1,
    "startDate": "2023-01-01",
    "endDate": "2023-12-31",
    "totalAmount": 11000.0,
    "averageAmount": 550.0,
    "paymentCount": 20,
    "currency": "EUR"
}
Beispiel Antwort (Create): 
{
    "id": 124,
    "fundId": 1,
    "startDate": "2023-01-01",
    "endDate": "2023-12-31",
    "totalAmount": 10000.0,
    "averageAmount": 500.0,
    "paymentCount": 20,
    "currency": "EUR"
}

6. ZahlungStatistik löschen

Beschreibung: Löscht eine ZahlungStatistik anhand der angegebenen ID.

Beispiel Anfrage: 
DELETE /api/statistics/123
Beispiel Antwort: 
HTTP/1.1 204 No Content

Fehlerbehandlung

Falsche Anfrage

Beschreibung: Wenn eine falsche Anfrage gestellt wird, wie eine nicht definierte Route, gibt der Server einen Fehler zurück.

Beispiel Anfrage: 
GET /api/nonexistent
Beispiel Antwort: 
HTTP/1.1 400 Bad Request
{
    "error": "Invalid endpoint or bad request"
}
